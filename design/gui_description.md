# GUI Design Ideas

1. **Side-by-side view**
   - Old file on the left, new file on the right.
   - Lines connected with colored lines.
   - Hover shows details.

2. **Flow diagram**
   - Boxes for groups of lines.
   - Arrows showing how lines moved or split.

3. **Heatmap**
   - X-axis = old file line numbers.
   - Y-axis = new file line numbers.
   - Color shows similarity.

4. **Mapping inspector**
   - Click a line to see its best matches and confidence score.
